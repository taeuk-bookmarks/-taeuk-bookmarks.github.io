Argo Smart Routing

한 번의 클릭으로 트래픽을 전 세계적 수준으로 가속화하세요

공용 인터넷은 콘텐츠 전달의 최선책이 아닙니다 - 그러나 네트워크 혼잡을 믿을 수 없어 로드 시간을 느려지게 하고 최종 사용자 경험 수준을 낮아지게 합니다.

Cloudflare 네트워크는 다릅니다. Cloudflare의 네트워크는 초당 평균 36백만 개의 요청을 라우팅하므로, Argo Smart Routing은 고유하고 유리한 위치를 점하여 실시간 혼잡을 감지하고 가장 빠르고 신뢰할 수 있는 네트워크 경로를 통해 웹 트래픽을 라우팅할 수 있습니다.

Argo를 사용하면 웹 콘텐츠의 속도가 평균 30% 빨라집니다.

빠른 로딩 시간

Argo는 최종 사용자 경험으로 가장 빠른 링크를 통해 웹 트래픽을 전달하여 눈에 띄게 속도가 빨라지고 웹 콘텐츠가 개선됩니다

신뢰도 향상

Argo의 네트워크 상태 라이브 보기는 혼잡을 피해 라우트하고 가장 안정적인 링크를 활용하여 가동 시간을 늘릴 수 있습니다

통합 보안

모든 Argo 트래픽은 Cloudflare 네트워크를 종단 암호화하여 웹 트래픽을 악의적 활동에서 보호해줍니다

비용 절감

Argo는 원본 서버에 대한 콘텐츠 요청을 최소화하여 대기 시간, 서버 부하 및 대역폭 사용을 절감합니다

“저희는 Argo를 통해 대기 시간을 최대 36%까지 줄였고 제한 시간 초과는 42%로 줄어든 것을 확인했습니다. 이는 곧 사용자에 대한 일치성이 증가하고 연결이 향상되었다는 것을 의미합니다.”

ALEX DUMITRIU

OkCupid CIO

동적 콘텐츠 전달

사용자의 50% 이상은 로드하는 데 2초 이상 걸리는 페이지를 보지 않습니다

최종 사용자 경험은 위치, 기기, 현재의 네트워크 상태에 관계 없이 몰입형, 대화형이어야 하고, 빨라야 합니다. Argo는 다음을 제공합니다.

  • 정체 방지: 실시간 네트워크 상태에 따른 라우팅 결정
  • 계층형 캐싱: 인접한 데이터 센터에서 콘텐츠를 제공하므로 캐시누락 비율, 서버 부하, 전체적 대기 시간 감축
  • TCP 최적화: TCP 연결이 끊이지 않아 캐싱할 수 없는 콘텐츠에 대한 대기 시간 감소
  • 글로벌 지원: 중국 포함 Cloudflare 전체 네트워크에 걸친 네트워크 라우팅

모든 트래픽을 가속화

패킷에 Argo Smart Routing을 적용함으로써 Cloudflare의 지능형 트래픽 가속이 이제 Magic Transit에서 작동합니다. Cloudflare 전역 네트워크를 이용해 모든 응용 프로그램을 DDoS 공격으로부터 방어하면서 대기 시간 감축, 성능 향상, 가용성 증대가 동시에 가능합니다.
